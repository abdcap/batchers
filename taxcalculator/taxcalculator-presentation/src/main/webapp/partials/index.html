Navigating the application

You can see that the application exists out of 4 parts:

  • Home is of course this page
  • In Employees you can see all the employees. By default, 12 employees are generated. You can also alter the number of employees here to real life figures (e.g. 100.000) but note that all existing data will be cleared, including the results of previous batch runs!
  • Run Job allows you to run the taxcalculation job for the latest 6 months.

Real life?

To simulate real life projects, the following was done:

  • Calculation of the tax will take a random amount of time that can be configured between a min and a max. This configuration can be done in the taxcalculator-application.properties
  • Paying the taxes is done via a webservice which fails for certain users. This configuration can be done in the taxcalculator-stubwebservice.properties
